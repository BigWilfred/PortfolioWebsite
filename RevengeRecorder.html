Revenge Recorder

(in development)

Personal project created using the structure of a general use loop sequencer. Challenged myself to design, construct and test this framework over a free weekend, with the goal of increasing my ability to work within set time restriction. Thoroughly enjoyed the challenge, having to make calculated sacrifices in order to deliver the framework at the desired time.

Current iteration of the system is an example of my unashamed ideation process; taking this sequencer framework to create a child's recorder looper, with the purpose of getting payback on noisy neighbours (I know, evil right?).

Growing up with three siblings, the ear piercing shriek of a recorder has been ever present and memorable. It has been profoundly enjoyable creating a website with a humorous purpose and I love the freedom it has provided.

User interface considers smooth playback, attempting to express a playful experience. Currently working on increasing the usability of the website, considering cross-browser performance and a more responsive UI. Presently hosted on Heroku, slowing load times as the dyno initialises after the request.
